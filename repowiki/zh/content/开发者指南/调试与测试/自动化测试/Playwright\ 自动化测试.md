## 安装与配置

### 安装 Playwright

```bash
# 安装 Playwright 依赖
npm install -D @playwright/test

# 安装浏览器
npx playwright install
```

### 配置文件

`playwright.config.js` 包含测试配置：

```javascript
module.exports = {
  testDir: './tests',
  fullyParallel: true,
  forbidOnly: !!process.env.CI,
  retries: process.env.CI ? 2 : 0,
  workers: process.env.CI ? 1 : undefined,
  reporter: 'html',
  use: {
    baseURL: 'http://localhost:3000',
    trace: 'on-first-retry',
    screenshot: 'only-on-failure',
  },
  projects: [
    {
      name: 'chromium',
      use: { ...devices['Desktop Chrome'] },
    },
    {
      name: 'firefox',
      use: { ...devices['Desktop Firefox'] },
    },
    {
      name: 'webkit',
      use: { ...devices['Desktop Safari'] },
    },
  ],
};
```

## 编写测试

### 基本测试示例

```javascript
const { test, expect } = require('@playwright/test');

test('homepage has title', async ({ page }) => {
  await page.goto('/');
  await expect(page).toHaveTitle(/TwinSight/);
});

test('user can login', async ({ page }) => {
  await page.goto('/login');
  await page.fill('[name="username"]', 'admin');
  await page.fill('[name="password"]', 'password');
  await page.click('button[type="submit"]');
  await expect(page).toHaveURL('/dashboard');
});
```

### 系统配置测试示例

```javascript
const { test, expect } = require('@playwright/test');

test.describe('系统配置面板', () => {
  test.beforeEach(async ({ page }) => {
    await page.goto('/system-config');
  });

  test('IoT配置面板显示正确', async ({ page }) => {
    // 等待配置面板加载
    await page.waitForSelector('[data-testid="iot-config-panel"]');
    
    // 验证面板标题
    const title = await page.textContent('[data-testid="panel-title"]');
    expect(title).toContain('IoT配置');
    
    // 截图对比
    const panel = await page.locator('[data-testid="iot-config-panel"]');
    expect(await panel.screenshot()).toMatchSnapshot('iot-config-panel.png');
  });

  test('可以修改MQTT配置', async ({ page }) => {
    await page.fill('[name="mqtt.host"]', 'mqtt.example.com');
    await page.fill('[name="mqtt.port"]', '1883');
    await page.click('button:has-text("保存")');
    
    // 验证保存成功提示
    await expect(page.locator('.success-message')).toBeVisible();
  });
});
```

## 运行测试

### 本地运行

```bash
# 运行所有测试
npx playwright test

# 运行特定测试文件
npx playwright test system-config.spec.js

# 运行特定浏览器
npx playwright test --project=chromium

# 调试模式（带UI）
npx playwright test --ui

#  headed 模式（显示浏览器窗口）
npx playwright test --headed
```

### 生成报告

```bash
# 运行测试并生成HTML报告
npx playwright test --reporter=html

# 查看报告
npx playwright show-report
```

## 截图对比测试

Playwright 支持视觉回归测试，通过截图对比检测UI变化。

### 更新快照

当UI有意图地发生变化时，更新快照：

```bash
npx playwright test --update-snapshots
```

### 快照存储

截图快照存储在 `tests/*.spec.js-snapshots/` 目录下，按浏览器和平台分类：

- `*-chromium-darwin.png` - macOS Chrome
- `*-firefox-darwin.png` - macOS Firefox
- `*-webkit-darwin.png` - macOS Safari

## CI/CD 集成

### GitHub Actions 示例

```yaml
name: Playwright Tests

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      
      - name: Setup Node.js
        uses: actions/setup-node@v3
        with:
          node-version: '18'
          
      - name: Install dependencies
        run: npm ci
        
      - name: Install Playwright
        run: npx playwright install --with-deps
        
      - name: Run tests
        run: npx playwright test
        
      - name: Upload report
        if: always()
        uses: actions/upload-artifact@v3
        with:
          name: playwright-report
          path: playwright-report/
```
