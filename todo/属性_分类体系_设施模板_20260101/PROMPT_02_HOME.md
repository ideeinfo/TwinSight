# Prompt 02: Home

```text
Page to generate: Home

Use the existing TwinSight UI design language.

Global application shell requirements:
- Add a top navigation bar with Home, Facilities, and Manage
- Home is active on this page
- Logo on the left, user menu on the right

Page purpose:
- This is the default page after login
- It is the system home page, not a public marketing landing page
- It helps users quickly choose and enter a facility

Main content sections:
- Welcome header
- Recent facilities section
- Facility card grid or compact list
- Quick actions area
- Optional lightweight system summary area

Facility selection requirements:
- Each facility item should support thumbnail, facility name, project name, owner, and address summary
- Each facility item should have a clear primary action to enter the facility workspace
- Show at least one quick way to create a new facility or go to Facilities management

Visual direction:
- Enterprise dashboard home page
- Compact and functional
- Avoid oversized marketing visuals
- Keep the page dense enough for real daily use
- Use cards only where useful, not everywhere

Interaction hints to reflect:
- Recent facilities are the most visually prominent actionable items
- The page should make it obvious that users can choose a facility from Home

Deliverable:
- Desktop-first layout
- Generate a realistic authenticated product home page for an engineering platform
```
