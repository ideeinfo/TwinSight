# 文档管理重命名功能 - 演示

我已在文档管理模块中实现了文件和文件夹的重命名功能。用户现在可以通过右键菜单或下拉菜单对项目进行重命名。

## 更改内容

### 前端实现
`src/components/DocumentManager.vue`

1.  **菜单更新**: 在列表视图和网格视图的下拉菜单中添加了“重命名”选项。
2.  **新对话框**: 添加了一个用于输入新名称的 `el-dialog`。
3.  **逻辑实现**:
    - `openRenameDialog`: 打开对话框并预填充当前名称。
    - `confirmRename`: 调用后端 API (`PATCH`) 更新名称，支持根据类型（文件/文件夹）调用不同的接口。

### 国际化
`src/i18n/index.js`

添加了新的翻译键：
- `rename`: 重命名
- `name`: 名称
- `renamePlaceholder`: 请输入新名称

## 验证结果

### 代码检查
- 确认后端 API `PATCH /api/v2/documents/:id` 和 `PATCH /api/v2/documents/folders/:id` 已存在并支持名称更新。
- 确认前端已正确调用上述 API。
- 确认输入框会在打开对话框时自动聚焦。

### 功能演示
用户点击文档或文件夹的“...”菜单，选择“重命名”，输入新名称后确认，列表会自动刷新显示新名称。
